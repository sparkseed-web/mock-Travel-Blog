# 模写コーディング　Travel-Blog 

## 概要
学習目的で制作した模写コーディング作品です。

## 使用技術
- HTML
- CSS

## 学んだこと意識したこと
- コードを書き始める前にサイトの全体のレイアウトを分析しパーツごとに分け、メモをとることから始める。
- デザインの共通点を見つけることでクラス名を付けやすくする。そうすることでコードの肥大化を防ぐこと。
-  文字の下にデザイン性の高い下線を引くために、`position: absolute` を用いたレイアウト手法を学びました。
-  画像を使わず、CSSのボーダーの重なりを利用して三角形（アイコン用）を描画する方法を習得しました。
